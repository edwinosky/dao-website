# ShieldDAO Front-end website

Front-end of SheildDao website created with react and next.js

# How to run

To run the project install packages first by :

```
yarn install
```

or

```
npm install
```

## Run Dev server

```
yarn dev
```

## Build Project

```
yarn build
```

Start after building

```
yarn start
```

Analyze bundle size

```
yarn analyze
```
